# Checklist for updates

## Versioning
```
X.Y.Z / 1.0.1

X = Major version
Y = Minor version
Z = Build updates
```

## Major/minor revisions
Changes to major or minor versions need to be logged in:
* `CHANGELOG.md`
* `nginx-s3-nomad.ini`
* Example job in `README.md`

## Shellcheck
Was `shellcheck` run on all applicable shell files?
